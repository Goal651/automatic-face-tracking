# CONTRIBUTING

Thank you for your interest in contributing to the Face Locking System! We welcome contributions from the community to help improve the performance, accuracy, and features of this project.

## How to Contribute

1.  **Report Bugs**: If you find a bug, please open an issue on GitHub. Include details about your environment and steps to reproduce the error.
2.  **Suggest Features**: Have an idea for a new feature? Open an issue to discuss it!
3.  **Submit Pull Requests**:
    - Fork the repository.
    - Create a new branch for your feature or bug fix.
    - Write clean, documented code.
    - Ensure your changes follow the existing project structure.
    - Submit a pull request with a clear description of your changes.

## Development Guidelines

- **CPU-First**: Maintain the project's commitment to being CPU-friendly. Avoid dependencies that require a GPU unless absolutely necessary.
- **Modularity**: Keep components (detection, alignment, embedding) separate and modular.
- **Testing**: Before submitting a PR, run the existing test scripts (e.g., `test_face_locking.py`) to ensure no regressions.

## Code of Conduct

Please be respectful and professional in all interactions. We aim to foster a collaborative and welcoming environment for everyone.
